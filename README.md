# Rose Lerato Melato | Portfolio Website

## Overview

This is my personal portfolio website, showcasing my journey from **Biochemistry to Software Development**, my technical skills, and my projects. The site includes a contact form with backend integration, allowing visitors to send messages directly to me.

The frontend is built with **HTML, CSS, and JavaScript**, while the backend uses **Node.js, Express, and MySQL**, including **email notifications** on form submission.

## Features
* **Responsive Design** – Mobile-first layout for desktop, tablet, and mobile devices.
* **Interactive Sections** – Hero introduction, learning journey timeline, skills showcase, and featured projects with GitHub links.
* **Backend Functionality** – Contact form submissions are validated, stored in a MySQL database, and trigger email notifications.
* **Smooth Navigation** – Mobile menu toggle, smooth scrolling for anchor links.

## Technologies
* **Frontend**: HTML5, CSS3, JavaScript
* **Backend**: Node.js, Express.js
* **Database**: MySQL
* **Email**: Nodemailer
* **Environment Management**: dotenv
* **Version Control**: Git, GitHub
